# Pokedex

A Pokedex App, made in flutter. This app is powered by [PokeApi](https://pokeapi.co/).

## Features
-  List of all pokemons
-  Search for pokemons
-  Pokemon details
-  Pokemon evolution chain
-  Pokemon moves
-  Pokemon stats
-  Pokemon types
-  Pokemon abilities
-  Move details
-  Ability details
-  Type details
